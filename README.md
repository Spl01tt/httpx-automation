# httpx-automation
This script automates httpx tool and can be used in Bug bounty to make information gathering a little bit faster.
Uses `-sc -cl -location -title -ip -td -server`

Usage:

```bash
chmod +x httpx.sh
./httpx.sh <subdomains-file>
```

If you want to change the output:

```
./httpx.sh <subdomains-file> custom.txt
```
